# NovaBank CRM — Salesforce

Gestión integral de tarjetas de crédito para persona natural.

## Estructura de Ramas
- `feature/novabank-core-implementation` — Apex, LWC, Aura, Flows
- `feature/novabank-rest-api` — REST API OAuth 2.0

## Versión
v1.0.0 — Implementación completa del ejercicio integrador

## Requisitos
- Salesforce Developer Edition
- VS Code + SFDX CLI
- Connected App configurada para OAuth 2.0

## Despliegue
sf project deploy start --source-dir force-app
